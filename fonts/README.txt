Download Orbitron font files and place them in this `fonts/` folder.

Recommended files (from Google Fonts or fonts.google.com):
- Orbitron-Regular.woff2    (preferred)
- Orbitron-Regular.woff     (optional fallback)
- Orbitron-Bold.woff2       (preferred)
- Orbitron-Bold.woff        (optional fallback)

How to get them:
1. Visit https://fonts.google.com/specimen/Orbitron
2. Select the desired weights (400 and 700) and click "Download family".
3. Unzip the downloaded archive and copy the .woff2/.woff files into this folder.

File layout expected by the site:
- cosite/
  - index.html
  - css/
    - style.css
  - fonts/
    - Orbitron-Regular.woff2
    - Orbitron-Bold.woff2
